# 🚖 Taxi Booking Service

A Spring Boot REST API project for managing taxi bookings.

## 🚀 Features

- Customer Registration
- Driver Registration
- Cab Registration
- Book a Cab
- Automatic Bill Calculation
- Trip Status Management
- Email Notification after Booking
- Exception Handling
- DTO & Transformer Pattern
- MySQL Database Integration

## 🛠️ Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- Lombok
- Java Mail Sender
- Postman

## 📁 Project Structure

- Controller
- Service
- Repository
- Entity
- DTO
- Transformer
- Exception
- Enum

## 📌 API Endpoints

### Customer
- POST /customer/add

### Driver
- POST /driver/add

### Cab
- POST /cab/add

### Booking
- POST /booking/book/customer/{customerId}
